<h1 align="center">  Central da Engenharia Química</h1>
  
É uma iniciativa independente com o objetivo de reunir, organizar e disponibilizar conteúdos relacionados à graduação em Engenharia Química. Este repositório busca ser um ponto de networking para estudantes, professores e interresados na área.
